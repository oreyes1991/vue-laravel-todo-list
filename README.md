# TODO list app with vue 3 and Laravel.

### How to run this project:

#### Step 1 run the core service:

```bash
    cd core-service/
```

- This will run the Docker environment

```bash
    ./vendor/bin/sail up
```

> [!WARNING]
> A Docker installation is required on your OS.

#### Step 2 run the web:

```bash
    cd web/
```

```bash
    yarn serve
```
